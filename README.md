# Chiranjeevani.AI

This is a Next.js application for Chiranjeevani.AI, an AI-powered education and safety platform for pharmacy students and professionals.

## Core Features

- **Pill Recognition and Information**: Search for drugs and get detailed AI-generated summaries.
- **Gamified Learning Modules**: Take interactive quizzes on various pharmaceutical topics.
- **Research Showcase Platform**: Get AI-powered suggestions for mentors and related research based on your interests.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
